# [프로젝트명: shhoban_piano]

## 1. 프로젝트 사전 점검

- 현재상황: 기존 홈페이지 유지보수 불가 상태
  - 원인 : 관련 소스코드, 기타 산출물 자료가 없음.
    - 닷홈에서 호스트 중이나, 접속 정보를 알 수 없어서 소스코드 접근 불가.

  - 참고사항1 : php 기반 웹 페이지
  - 참고사항2 : 도메인 주소 (www.hobanmusic.co.kr)
  - 참고사항3 : 도메인 주소는 추후 변경 예정.
